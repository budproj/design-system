# Installing our packages

After completing all the required steps at the [prerequisites section](?path=/story/introduction-getting-started--page#prerequisites) at the [Getting Started](introduction-getting-started--page), you can proceed to the package you're trying to install at this page and execute all the installation steps.

## Core

### 1. Install the required peer dependencies

To install them, run the following command:

``` sh
yarn add react@17.0.0 theme-ui@0.3.1
```

### 2. Install this package in your application

To install it, run the following command:

```sh
yarn add @bud/core
```

## Tokens

### 1. Install this package in your application

To install it, run the following command:

```sh
yarn add @bud/tokens
```
